Fattura accompagnatoria

Agile Business Group

Questo modulo permette di stampare la fattura accompagnatoria. La fattura accompagnatoria è un documento valido in Italia che unisce il DDT e la Fattura.

Dal menu Contabilità, Fatture Clienti selezionare una fattura.

Aggiungere le informazioni circa la consegna nel tab "informazioni spedizione"

Per stampare la fattura accompagnatoria, dal pulsante STAMPA selezionare Fattura accompagnatoria

Nel report PDF della Fattura sono aggiunti tutti i dati relativi alla spedizione

Setup

Le tabelle dati di spedizione sono precaricate con dati di default. Dal menu Inventario, Configurazione, dati DDT.
